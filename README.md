# WorkReport
 `WorkReport` 是记录工作日志的简易系统，主要用于以事件的形式记录工作日志
#依赖
* node v14.15.1+
* npm 6.14.8+
* golang 1.16+
* git


# 自行编译
[![Build Status](https://travis-ci.com/YouCD/WorkReport.svg?branch=master)](https://travis-ci.com/YouCD/WorkReport)

* 编译前先运行`build.sh`会自动编译npm
```shell
sh build.sh
```

* linux

```shell
make build-linux
```
* darwin
```shell
make build-darwin
```

* windows
```shell
make build-win
```
